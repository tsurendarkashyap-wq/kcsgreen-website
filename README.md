# KCS Green Energy Solutions Website

A modern responsive Next.js + Tailwind website for www.kcsgreen.com.

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Recommended: Vercel. Connect this folder/repository and set the production domain to www.kcsgreen.com.

## Pages/sections included
- Home
- About
- Services
- Capabilities
- Projects
- Clients
- Contact form UI

Note: the contact form is frontend-only. Connect it to Formspree, Zoho, Google Forms, or a custom backend before going live.
